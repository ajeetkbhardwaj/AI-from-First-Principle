
# Questions and Answers

## Q: What is Markdown?
**A:** Markdown is a lightweight markup language with plain text formatting syntax.

## Q: How do I create a list in Markdown?
**A:** You can create an unordered list using asterisks or hyphens:
- Item 1
- Item 2
